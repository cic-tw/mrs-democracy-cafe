# 蒂瑪小姐咖啡館

一杯咖啡的時間，可以聊生活，可以聊八卦

帶著裝著稻穗的玻璃瓶鑰匙圈，為了工作來到  
台北。他說：如果可以，我想留在家鄉工作。

她從小在台北長大……  
習慣搭捷運上下班，一直都與家人同住  
她說：好想有自己的房子，但買不起。

來聊聊吧！
